Steps to run the application :
- `git clone https://github.com/damienassany/weathy.git`
- `cd weathy && yarn install`
- `yarn start`

To run the unit tests :
`yarn test`

To run the storybook :
`yarn storybook`

To build the application :
`yarn build`
